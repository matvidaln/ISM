
1- para clonar un repositorio:

 git clone git@github.com:matvidaln/ISM.git


2- para descargar la ultima version desde el repositorio: dentro del directorio que esta sincronizado, hacer 

 git pull

3- para ver como se compara la version local con la del repositorio:

 git status

4 - para agregar archivos:

 git add filename.txt

ejemplos:
git add tareas/* 

5 - para comprometer cambios 

 git commit -am "comentario para describir cambios"

ejemplos:
git commit -am "con tareas"

6- para subir cambios 

git push
